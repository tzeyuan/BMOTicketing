BMO Ticketing is a full-stack online ticketing system that allows users to browse events, purchase tickets, participate in community discussions, and buy event merchandise. It also includes an admin management panel for creating and managing events, users, and sales reports.

Features
User Portal
- Browse and search for events
- View event details and ticket availability
- Virtual waiting room before ticket purchase
- Select ticket type and quantity
- Simulated payment system
- Community discussion threads for each event
- Merchandise browsing and shopping cart

Admin Panel
- Create new events
- Edit and delete existing events
- Set ticket types and ticket limits
- Upload event images
- View registered users
- Monthly sales report and ticket sales summary
